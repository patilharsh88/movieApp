Movie 🎬 App

A simple and modern movie browsing application that lets users discover trending films, search for titles, and view detailed information about each movie.

🚀 Features
- Search for movies by title
- View trending and popular movies
- See movie details (rating, release date, overview, poster)
- Add movies to favorites (optional feature)
- Responsive and user-friendly UI
- Tech Stack

Frontend: HTML, CSS, JavaScript ,React.js
Backend: Will be in SpringBoot
API: TMDB (The Movie Database) API
Styling: CSS
